# Media service

## Description

Service for storing media files

## Installation

> With yarn

```bash
yarn
```

> With npm

```bash
npm i
```

## Running the app

> With yarn

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

> With npm

```bash
# development
$ npm start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
